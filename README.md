AI-Driven Smart Irrigation and Crop Advisory System
Overview
This project is an IoT-based smart irrigation system that monitors environmental conditions such as soil moisture, temperature, humidity, and water level to provide intelligent irrigation recommendations. The goal is to help farmers optimize water usage and improve crop productivity.
Features
Real-time monitoring of soil moisture, temperature, and humidity
Water tank level monitoring
Automatic irrigation decision logic
Dashboard interface for farmers
Chatbot-based irrigation suggestions
Hardware Used
ESP32 DevKit V1
DHT22 Temperature and Humidity Sensor
Soil Moisture Sensor
HC-SR04 Ultrasonic Distance Sensor
Tech Stack
Frontend: HTML, CSS, JavaScript
Charts: Chart.js
